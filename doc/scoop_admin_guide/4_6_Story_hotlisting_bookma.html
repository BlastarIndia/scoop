4.6 Story hotlisting/bookmarking

Scoop offers a `hotlist' feature which allows users to mark stories they would like to watch. Hotlisted stories are listed in a menu in the sidebar on every page via the hotlist box, for easy access, and display the number of new comments (4.8.2) since the story was last viewed. The hotlist box disappears entirely if no stories are hotlisted, unless the hotlist_flex or one of its variants (available from the Scoop Box Exchange: B) is used.

The hotlist add/remove link is placed in the story_summary block using the special key |hotlist|. The actual text (or image) of the hotlist link is set in the blocks hotlist_link and hotlist_remove_link.

The hotlist perm (A.12.32) determines who can use the hotlist function and see the hotlist add/remove links.

Note that story hotlisting is distinct from diary subscriptions (B.3.1), although many people use `hotlist' to describe them both; story hotlisting does not notify you of new stories, it allows you to track specific existing stories.
