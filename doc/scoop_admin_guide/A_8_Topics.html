A..8 Topics

The Topics Admin Tool is where you edit the topics and associated topic images. To use this Admin Tool, you must have the edit_topics perm (A.12.28) active for your group.

To turn topics on and off, see the variable use_topics.

On the Topics admin page, all topics are listed with their images to the right of the page, and the topic detail form for the currently selected topic is to the left. Clicking on a topic will bring its details into the form to the left.

TID
The topic ID, a short alphanumeric string used to uniquely identify the topic.
Dimensions
Used when displaying the image in a story. Enter the pixel width in the first box and the pixel height in the second.
Alt Text
Used both as the alt attribute when displaying the topic icon and also as the descriptive name in the select box when a new story is being posted.
Image Name
The filename of the topic image, with no path information. Scoop assumes that all topic images are in the subdirectory named in the variable topics of the directory named in the variable imagedir and will prepend that path to whatever image name you supply.

If you are editing an existing topic, an additional option is shown, used if you are deleting the topic. Another topic should be selected, and all stories using the deleted topic will be reassigned to the topic selected here.

The ``Write'' button either saves changes or creates a new topic, depending on whether the topic ID exists already or not. The ``Delete'' button deletes the topic and reassigns stories as required.
