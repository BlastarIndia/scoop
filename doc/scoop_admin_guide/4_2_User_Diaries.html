4.2 User Diaries

Every user can post their own musings in their own section. Basically, it's as if every user has his own personal weblog on your site. The Diaries section can be used for anything the users want to use it for: getting to know each other, venting, or requesting feedback on a draft of an article they're working on.

The Diary section is set up so that all users submissions will auto-post to section (see section 4.19.3 and appendix A.9 for details). This means that Diaries are not sent through the queue. Apart from that, diaries have all the same features as stories and are edited in the same way. Diaries are not shown on the ``Everything'' index page by default, but this can be changed by settings the sections_excluded_from_all variable. The RDF feed (see section 4.5 for RDF information) also uses this variable to control what stories are shown in the RDF file.
