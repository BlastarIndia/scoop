A..4 New Poll

The New Poll Admin Tool brings you to a blank poll form, where you can enter a poll question and answers independant of a story. To use this Admin Tool, you must have the edit_polls perm (A.12.24) active for your group.

The ``Poll Question'' and ``Poll Answers'' fields are where you input the question and possible answers, respectively. The number of available poll answer fields is configurable using the variable poll_num_ans.

The checkbox ``Make Main'' determines whether the poll will be made into a front/section page poll when saved, replacing the previous poll, or whether it will just be saved. There is no preview function.

If the variable allow_ballot_stuffing is turned on, a second column of textboxes appears beside the ``Poll Answers''. The administrator can put the initial number of votes for each possible answer in those textboxes. This also allows you to edit an existing poll and change the number of votes, or reset them all to zero.
