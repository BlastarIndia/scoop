A..19 Log

The Log Admin Tool is where you view and manage the activity logs. The first page of the Log Admin Tool is a simple list of links to the different logged item views and actions you can take.

The logged item views are:

View list of all logged records
This link displays a list of all log records recorded to date. The list includes information such as the action taken, the item acted upon, the person who performed the action, and the time and date. If extended logging is enabled, a `More' link leads to extra details for each specific item. This list includes the items shown in the three views below as well as any others.
View list of deleted comments
This link displays a list of all log records for comment deletion. The details are as above.
View list of deleted stories
This link displays a list of all log records for story deletion. The details are as above.
View list of updated stories
This link displays a list of all log records for story updates. The details are as above.

The actions are:

Clear logfile
This link clears all items from the log, and adds a log item indicating that the log has been cleared, by whom, and when. The log item generated by this link cannot be avoided, even if all other logging functionality is turned off.
Change logging level
This link brings you to the Site Controls Admin Tool (A.6) with the variable use_logging displayed. From there you can set the log level to none, basic, or extended.
Enable all logging hooks
This link enables all hooks that use the function log_activity. Hooks are defined in the Hooks Admin Tool (A.18).
Disable all logging hooks
This link disables all hooks that use the function log_activity.
Customize logging hooks
This link brings you to the Hooks Admin Tool (A.18) where you can edit any of the logging or other hooks defined there.

For more information on setting up activity logs, see section 4.23
